Solar System Project

Lancaster University

This project was completed in November-December 2018 in approximately 3-4 weeks. This was the first project I had ever developed in Python and was only a few weeks after learning Python for the first time. I was tasked with creating a computer simulation of a complex physical system and to write a scientific report on my results.

The simulation modelled the major bodies of the solar system (the Sun, the planets and Pluto) gravitationally by solving the ordinary differential equations associated with the system. I solved these ordinary differential equations using iterative methods such as Euler's method and the Runge-Kutta methods. I wanted to compare different iterative methods to one another, to see which methods were more accurate or more time-consuming than others.

The simulation had 3-dimensional position and velocity vectors for each body. Using the positions and velocities, the simulation calculates the acceleration for each body due to gravity acting from all the other bodies in the system. Using this acceleration, I could calculate the next position and velocity vectors for the following timestep.

I measured the energy, momentum and angular momentum to see if they were conserved. If they were conserved, then the system should be following Newtonian phyiscs and the simulation would be accurate. I then wrote an 11-page report on my findings.
