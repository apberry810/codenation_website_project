Wildfire Group Project

Lancaster University

This project was completed in November-December 2018 in approximately 3 weeks. This was the first group project I had completed in Python. We were tasked with building any kind of simulation we wanted within a cellular automata framework. Cellular automata is a model of computation. A famous example of cellular automata would be Conway's game of life.

We agreed to build a wildfire simulation based on a specific case study in Australia on the Black Saturday fires in February-March 2009.

Two of the members of our group had no experience coding at all, so me and one other person in our group also had to teach the group members with no experience Python as well as complete the project in time.

I was responsible for obtaining the wind data and for coding the population model we worked on. The population would occupy cells, if there were a fire in a nearby cell that population would then move away from the fire cell in a random direction. We would measure the amount of casualties for each run of the simulation and how that number changed depending on when they started to move away from the fire.

We wrote a 42-page scientific report on our findings. Unfortunately the code is not available for this project, but if you would like to read the scientific report, follow this github link or click on the image above.
